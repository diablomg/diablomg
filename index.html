How you update the site

  1. Drop your new comic image into the assets folder.
  2. Copy one comic object in script.js.
  3. Change the title, date, series, month, excerpt, and image path.
  4. Put the newest comic first in the array so it becomes featured.
  5. Commit and push to GitHub Pages.
